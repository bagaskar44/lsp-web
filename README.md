# Sistem Informasi Perpustakaan Arcadia (Drive-Thru)

Aplikasi web berbasis Flask untuk manajemen peminjaman buku dengan sistem Drive-Thru.  
Memiliki dua modul utama: **Peminjam** (Member) dan **Admin**.

---

## Fitur Utama

### **Peminjam**
- Landing page
- Registrasi dan Login
- Katalog Buku
- Keranjang
- Tiket Kode Pinjam (ditunjukkan saat Drive-Thru)

### **Admin**
- Dashboard Pesanan
- Manajemen Stok / Edit Pesanan
- Persetujuan (Approval)
- Konfirmasi Pengembalian

---

## Teknologi

- **Backend**: Python Flask  
- **Database**: MySQL (flask_mysqldb)  
- **Frontend**: HTML5, Bootstrap 5
